# Skill contracts

- `detect-buying-signals`: accept normalized events from research, first-party systems, feeds, or CSV.
- `validate-signal`: verify attribution, timing, evidence fields, and duplicate fingerprints.
- `score-account-intent`: apply type weight, decay, reliability, confidence, provenance, reinforcement, and ICP fit.
- `map-buying-committee`: translate active signals into affected functions and personas.
- `write-evidence-based-outreach`: use the strongest verified event and phrase implications as possibilities.
- `refresh-signal-watchlist`: rescore events over time and remove expired signals from active account intent.
